Welcome to the Self-Driving Car Simulation

This project simulates a self-driving car using a neural network. The car uses sensors to detect obstacles and makes decisions based on the neural network's outputs. Initially, the car might not perform well, but through trial and error, you can train the model to improve its performance.

Here's what you can do:

  • Adjust the number of cars and the variation in weights to train the model.
  • Save the best car's brain if it performs well.
  • Discard the saved brain to start fresh.
  • Add more traffic to test the car's ability to navigate through obstacles.
  • Reset the simulation if the car crashes or gets stuck.
  • Change the number of lanes to see how the car adapts to different road conditions.

Car Count

Simulation of model learning by choosing the traffic and making your own bestCar through Trial and Error. Set the number of cars (1-100) to train the model also the variation in weights (0-1) wrt the bestCar. This will reset the simulation with the specified number of cars. Then follow the steps for Model Training

Model Training

Save the best car's brain to improve the model over time(in case a car fails to navigate through traffic then save the best play of the alive car/ destroyed car). Discard the saved brain if you want to start fresh.

Traffic Control

Add more traffic cars(randomly generated) to test the self-driving car's ability to navigate through obstacles.

Lane Count

Set the number of lanes for the road. This will reset the simulation with the specified number of lanes.

Reset Simulation

Reset the simulation if the car crashes or gets stuck.
